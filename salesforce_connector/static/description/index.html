ODOO 10.0 Salesforce Connector

Sync Salesforce with odoo using Salesforce Connector

Overview

ODOO is a fully integrated suite of business modules that encompass the traditional ERP functionality. Odoo integration with Salesforce gives us power to sync customers, products, sale orders and opportunities.

Key Features And Functions

  • Import Customers from Salesforce to Odoo
  • Import Sale Orders from Salesforce to Odoo
  • Import Opportunities from Salesforce to Odoo
  • Export Product to Salesforce from Odoo

Prerequisites

  • Install "Simple-salesforce" python library.
  • pip install simple_salesforce.

SalesForce Account

  • Create a Salesforce account salesforce login
  • After login click on user icon on top right corner of the screen and select settings. In settings you can see a menu-bar on left side. In menu bar click Reset My Security Token. It will send an email to your salesforce mail id which contains security token.

SalesForce Credentials

  • Go to Settings -> Users and click on a user and insert the Salesforce details as given below:
  • Note: You can also verify the credentials by clicking on Test Credentials button and save it.

Sync data with SalesForce

  • After test and save credentials, go to Sales force module. You can see salesforce-odoo connector home screen with options to import customers, sales orders, opportunities and export products . After sync complete you can view sync history in CONNECTOR History Detail.

Sale Orders

  • Check import sale orders and click sync button
  • You can see sale orders imported in ODOO in sales module
  • NOTE: Sale orders with no customer will not be imported to Odoo, as customer is required field in Odoo sale order.

Opportunities

  • Check import opportunities and click sync button
  • You can see opportunities imported in ODOO in sales module

Customers

  • Check import customers and click sync button
  • You can see customers imported in ODOO in sales module

Products

  • Check import products and click sync button
  • You can see products imported in ODOO in sales module

Scheduler to Sync Opportunities

  • Go to settings and enable developer's mode. Now go to settings -> technical -> automation -> scheduled actions. In list select Opportunity Scheduler. Here you can change interval number and interval unit(e.g minutes, hours, days etc) . This tells that after how much time opportunities should sync.
